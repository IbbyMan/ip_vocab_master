# 技能背单词 | IP Vocab Master

> 用IP趣味联想，轻松背单词！

## 📖 项目介绍

技能背单词是一款通过IP（动漫/游戏/影视作品）趣味联想来帮助记忆英语单词的应用。将枯燥的背单词变成有趣的IP场景联想！

## ✨ 功能特点

- 🎨 支持多种热门IP（哈利波特、王者荣耀、宝可梦、漫威、甄嬛传、老友记）
- 🔧 支持自定义IP
- 🧠 AI生成的词根词缀拆解+IP记忆联想
- 📚 生词本功能，支持导出/导入CSV
- 💾 本地存储，数据持久化

## 🚀 快速开始

### 前置要求
- Node.js

### 安装运行

```bash
# 1. 安装依赖
npm install

# 2. 在 .env.local 中设置 API_KEY

# 3. 启动开发服务器
npm run dev
```

### 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 🛠️ 技术栈

- React 19
- TypeScript
- Vite
- Google Gemini API

# IP Vocab Master

> Memorize English words with fun IP associations!

## 📖 About

IP Vocab Master is an app that helps you memorize English words through fun IP (anime/game/TV show) associations. Turn boring vocabulary learning into interesting IP scene associations!

## ✨ Features

- 🎨 Support for multiple popular IPs (Harry Potter, Honor of Kings, Pokemon, Marvel, Empresses in the Palace, Friends)
- 🔧 Custom IP support
- 🧠 AI-generated etymology breakdown + IP memory associations
- 📚 Vocabulary book with CSV export/import
- 💾 Local storage for data persistence

## 🚀 Quick Start

### Prerequisites
- Node.js

### Installation & Running

```bash
# 1. Install dependencies
npm install

# 2. Set API_KEY in .env.local

# 3. Start dev server
npm run dev
```

### Build & Deploy

```bash
# Build for production
npm run build

# Preview build result
npm run preview
```

## 🛠️ Tech Stack

- React 19
- TypeScript
- Vite
- Google Gemini API
